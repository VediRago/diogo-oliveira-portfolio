# Project: Fragments

## Overview

*Fragments* is a long-form narrative project about separate lives, recurring outcomes, emotional continuity, and hidden structure.

The project combines character-driven storytelling with layered narrative design.

It is currently being developed as a story world, writing project, and narrative design portfolio case study.

## Design Focus

The project focuses on how a reader's understanding evolves over time.

Instead of relying only on final reveals, *Fragments* gives the reader partial explanations that later expand.

Key design questions:

- How can mystery reward attention without withholding too much?
- How can character arcs work emotionally before they become structural clues?
- How can repeated images and behaviors create later recognition?
- How can different fragments feel separate while still belonging to one system?

---

## Narrative Systems

Current systems include:

- Emotional act map
- Bridge index
- Visual motif bridges
- Misdirection bridges
- Condition / virus bridge
- People-before-system rule
- Layered narrative system
- Unity / ending bridges

These systems are used to keep the project coherent without forcing exposition into the story.
