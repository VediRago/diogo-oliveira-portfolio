# Resolution

← Back to [Cosmology Reference](Cosmology_Reference.md)

Resolution is the normal process by which [The Condition](The_Condition.md) completes.

For fragments connected to this underlying self, resolution usually means early death.

This is not punishment. It is normal system behavior inside the hidden architecture. The story should not frame it as evil destiny. It should feel like a pattern that characters suffer before anyone understands what it is.

Resolution is the outcome that [Unity](Unity.md) disrupts. After Kael's anomaly spreads through associated fragments, the old resolution pathway stops functioning in the same way.
