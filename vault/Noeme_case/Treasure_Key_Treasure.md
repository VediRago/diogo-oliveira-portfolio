# Treasure Key Treasure

A discovery should be rewarding when first found.

Later, the same discovery may unlock a larger layer.

```text
Treasure
↓
Key
↓
New treasure
↓
New key
```

This keeps earlier material valuable.

## Core Rule

A scene, character, image, or clue can be:

- a treasure by itself
- a key to another layer
- both at the same time

The first value matters.

The later value should enrich it, not consume it.

## Fragment Value

Lucreta is not only setup for later events.

Sera is not only a mechanism for Unity.

Each fragment needs its own emotional value before it becomes structurally useful.

This supports the [[Layered Narrative System]] because the audience can care before they understand the larger structure.
