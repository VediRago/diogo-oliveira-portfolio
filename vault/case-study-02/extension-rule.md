# Extension Rule

The modular method should only grow when the material requires it.

At the character level, the base chain is enough:

**Experience → Emotion → Reason → Behavior**

This explains why a person acts.

When repeated behavior begins shaping the mood of a place, culture, or faction, the chain can extend into atmosphere.

When atmosphere survives across time and becomes ritual, law, record, myth, institution, or inherited belief, the chain can extend into history.

When that history creates pressure that can be acted on, the chain can extend into quest logic.

The rule is simple:

**Do not extend the chain unless the next layer is earned.**

This keeps the method modular. A character does not always need history. A place does not always need quest logic. A faction does not need every layer explained at once.

The method stays useful because it grows only when the design problem asks for the next layer.
